# Dubai Car Mechanic & Battery Replacement Website Template

A premium, high-converting, mobile-responsive landing page template custom-built for **Dubai Car Mechanic & Car Battery Replacement** (Nad Al Sheba 1, Dubai). 

This template is designed to be used for cold outreach, emails, or portfolio demonstrations to show the client how a premium, modern web presence can improve their lead generation and customer bookings.

## 🚀 Key Features

* **Premium Dark Automotive Aesthetic:** Styled using a sleek dark theme, energetic crimson red accents, and clean modern typography (**Outfit** and **Plus Jakarta Sans**).
* **Interactive Dispatch & Quote Calculator:** Client-side JavaScript widget that dynamically calculates starting prices and dispatch arrival times depending on the selected car brand and service.
* **Direct WhatsApp Integration:** 
  * The estimator dynamically builds pre-filled WhatsApp message URLs.
  * The rapid dispatch form validates inputs and formats them into a clean template to send straight to the dispatcher's WhatsApp.
* **On-Site Proof Photo Grid:** Hides pixelation of low-resolution images by placing them inside styled "Live Report" frames.
* **Google Maps Review Slider:** Responsive carousel displaying actual verified Google reviews (rating 5.0, 24 reviews) including screenshots, reviewer details, and replies.
* **Mobile-First Layout:** Includes sticky floating action CTAs at the bottom of mobile viewports so stranded drivers can immediately call or WhatsApp with one click.

## 📂 File Structure

* `index.html`: Core markup and calculator logic.
* `style.css`: Clean, modular, responsive vanilla styling.
* `brand given pics on google/`: Contains the business's actual service photos.
* `cutomer review pics/`: Contains the Google Maps review screenshots.

## 🔧 How to Customize

All contact details and pricing calculations are kept clean and easily editable:
1. **WhatsApp & Phone Link:** Update the phone number `+971 56 532 2547` in the `tel:` and `wa.me/` URLs.
2. **Pricing Rules:** Modify the `priceTable` base prices and `brandModifications` multipliers in the `<script>` tag of `index.html` to adjust the estimator calculations.
3. **Dispatch Area List:** Edit the grid items under `<section id="coverage">` to change the list of serviced neighborhoods.
